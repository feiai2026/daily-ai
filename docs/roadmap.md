# Roadmap

## v1：公开源中文 AI 日报

- GitHub 和 Hacker News 候选采集
- 官方公司与独立作者人工/Agent 核验
- 客观指标分层筛选
- 固定中文模板
- 本地质量检查

## v1.1：稳定性

- RSS 优先抓取官方博客
- 跨日去重
- GitHub API rate limit 提示
- 可配置时区与报告日期

## v1.2：更多量化信号

- 保存每日 GitHub 快照
- 计算真实 star 增量
- 识别新仓库增长速度
- 增加 release 与 commit 活跃度

## v2：轻量智能排序

- 客观阈值完成第一轮筛选
- AI 只负责相关性判断、摘要和主题聚类
- 增加周报与趋势变化报告

## 暂不包含

- 付费 X API
- 依赖登录态的抓取
- 第三方平台发布
- 内置定时调度
